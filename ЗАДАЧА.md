# Домашнее задание: Управление студентами с Redux Toolkit и сервером

## Цель работы

Создать полноценное React-приложение с Redux Toolkit для управления студентами и Express-сервером с PostgreSQL. Вы научитесь:
- Работать с Redux Toolkit (createSlice, configureStore)
- Выполнять асинхронные запросы к серверу
- Реализовывать CRUD операции (Create, Read, Update, Delete)
- Управлять формами с режимом редактирования
- Работать с REST API

## Описание приложения

Нужно создать приложение для управления студентами, где можно:
- ✅ Просмотреть список всех студентов
- ➕ Добавить нового студента (имя, email, группа, возраст)
- ✏️ Редактировать существующего студента
- ❌ Удалить студента
- 🔄 Загружать студентов с сервера при запуске приложения

**Важно:** Всё состояние (список студентов, режим редактирования) должно храниться в Redux Toolkit!

## Структура данных

### Студент (Student)

```javascript
{
  id: 1,                              // number (генерируется БД)
  name: "Ахмед Магомедов",            // string
  email: "ahmed@example.com",         // string (уникальный)
  student_group: "ФП-301",            // string
  age: 20,                            // number (16-100)
  created_at: "2024-01-15T10:30:00"   // timestamp
}
```

## Задачи для выполнения

## Часть 1: Backend (Сервер)

### Задача 1: Создать базу данных

1. Откройте PostgreSQL (через pgAdmin или psql)
2. Создайте базу данных `students_db`
3. Выполните SQL скрипт из файла `СОЗДАНИЕ ТАБЛИЦЫ.sql`

**Проверка:**
```sql
SELECT * FROM students;
```
Должна вернуться таблица с 5 тестовыми студентами.

---

### Задача 2: Реализовать API эндпоинты (server/index.js)

В файле есть заготовки для всех эндпоинтов. Вам нужно дописать логику.

#### 2.1. GET /students
- Получить всех студентов из БД
- Сортировать по id

#### 2.2. POST /students
- Добавить нового студента
- Валидация: name, email (с @), student_group, age (16-100)
- Вернуть созданного студента со статусом 201

#### 2.3. PUT /students/:id
- Обновить существующего студента
- Валидация полей (как в POST)
- Вернуть обновленного студента

#### 2.4. DELETE /students/:id
- Удалить студента по id
- Вернуть id удаленного студента

## Часть 2: Frontend (Клиент с Redux Toolkit)

### Задача 3: Создать slice для студентов (studentsSlice.js)

**Структура initialState:**
```javascript
{
  items: [],            // массив студентов
  editingStudent: null  // студент для редактирования (или null)
}
```

**Reducers, которые нужно создать:**
- `setStudents` - установить список студентов
- `addStudent` - добавить студента в начало
- `updateStudent` - обновить студента и завершить редактирование
- `deleteStudent` - удалить студента по id
- `startEditing` - начать редактирование
- `cancelEditing` - отменить редактирование

---

### Задача 4: Создать store (store.js)

Используйте `configureStore` из Redux Toolkit.

---

### Задача 5: Подключить Redux к React (main.jsx)

Оберните приложение в `Provider`.

---

### Задача 6: App.jsx - загрузка студентов при монтировании

- При монтировании компонента сделать GET запрос к `/students`
- Полученных студентов отправить в Redux через `dispatch(setStudents(students))`

---

### Задача 7: StudentForm.jsx - форма добавления/редактирования

- Получить `editingStudent` из Redux через `useSelector`
- Если `editingStudent` не null - заполнить форму данными для редактирования
- При сабмите формы:
  - Если редактируем: отправить PUT запрос, затем `dispatch(updateStudent(updatedStudent))`
  - Если добавляем нового: отправить POST запрос, затем `dispatch(addStudent(newStudent))`
- Кнопка "Отмена" при редактировании должна вызывать `dispatch(cancelEditing())`

---

### Задача 8: StudentList.jsx - список студентов

- Получить список студентов из Redux
- Отобразить студентов в виде карточек
- Для каждого студента предоставить кнопки:
  - **Редактировать**: вызвать `dispatch(startEditing(student))`
  - **Удалить**: отправить DELETE запрос, затем `dispatch(deleteStudent(id))`

## Частые ошибки

### ❌ Ошибка 1: Пытаюсь использовать классический Redux
**Решение:** Используйте `createSlice` и `configureStore` из Redux Toolkit

### ❌ Ошибка 2: Форма не заполняется при редактировании
**Решение:** Не забудьте useEffect с зависимостью от editingStudent

### ❌ Ошибка 3: Не могу получить данные из store
**Решение:** Проверьте путь в useSelector: `state.students.items`
